Dokumentasi Uinsu Web Framework

Instalasi

Offline
  • Download Source
  • Ekstrak, kemudian letakkan di folder webserver lokal, hosting.
Online (Menggunakan git bash)
  • git clone https://github.com/haxorsprogramming/Uinsu-Web-Framework
Lakukan setting awal di /engine/rule/base.php, dan perhatikan beberapa hal berikut
  • Pastikan mode re-Write-Engine aktif di server anda
  • Jika di upload di hosting, perhatikan url dengan http dan https berbeda dalam penanganan request
